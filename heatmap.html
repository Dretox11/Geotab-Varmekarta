Detta verktyg hjälper dig att visuellt analysera din fordonsflottas rörelsemönster. Identifiera snabbt var dina fordon tillbringar mest tid eller var specifika regelbrott (t.ex. fortkörning eller tomgångskörning) sker mest frekvent.

Inställningar

Platshistorik Avvikelse

Så här gör du

  1. Välj om du vill titta på Platshistorik eller Regelbrott.
  2. Markera önskade fordon i listan.
    Tips: Använd MyGeotabs globala gruppfilter i toppmenyn för att automatiskt sortera fram rätt fordon!
  3. Välj tidsperiod.
  4. Klicka på Visa Värmekarta.

Tips: Håll in CTRL (eller CMD på Mac) för att välja flera enskilda fordon i listan. För att snabbt markera alla fordon, klicka på ett av dem och tryck sedan CTRL + A.
