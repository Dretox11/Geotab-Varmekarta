Detta verktyg hjälper dig att visuellt analysera var specifika Avvikelser (t.ex. fortkörning, tomgångskörning eller skarpa svängar) sker mest frekvent.

Inställningar

Så här gör du

  1. Välj vilken regel du vill analysera i rullgardinsmenyn.
  2. Markera önskade fordon i listan.
    Tips: Använd MyGeotabs globala gruppfilter i toppmenyn för att automatiskt sortera fram rätt fordon!
  3. Klicka på Tidsperiod för att välja önskat tidsspann.
  4. Klicka på Visa Värmekarta.

Tips för att välja fordon: Håll in CTRL (eller CMD på Mac) för att välja flera enskilda fordon i listan. För att snabbt markera alla fordon, klicka på det översta, scrolla ner, håll in SHIFT och klicka på det nedersta.
